# Promptly — 提示词拼合器

一款专为 AI 绘画 / 大模型提示词创作设计的可视化拼合工具。将零散的提示词片段组合、排序、翻译，一键输出完整的提示词文本。

## 特性

- 🧩 **标签块编辑** — 输入自动切分为独立标签块，可单独启停、拖拽排序、跨片段移动
- 📝 **纯文本模式** — 自由输入，适合长描述和负面提示词
- 🌐 **一键翻译** — 支持中文翻译为英文，Google 直连或自定义大模型 API
- 🗂 **多标签页工作区** — 最多 5 个独立工作区并行编辑
- 💾 **快照与预设** — 保存历史版本，复用常用片段
- 📦 **导入/导出** — JSON 格式备份，合并式导入
- 🌓 **深色/浅色主题** — 跟随系统或手动切换
- 🔐 **多用户支持** — 注册登录、JWT 认证、数据隔离
- 🐳 **Docker 部署** — 一键部署，支持 amd64 + arm64

## 快速上手

### 界面布局

Promptly 采用**左右双栏布局**：

| 区域 | 位置 | 功能 |
|------|------|------|
| 标签栏 | 左上方 | 切换/管理工作区标签页 |
| 编辑区 | 左半区 | 编辑提示词片段 |
| 预览区 | 右半区 | 实时查看拼合结果 |
| 工具栏 | 右上方 | 分隔符、翻译设置、主题、保存、历史 |

### 基本流程

1. 登录或注册账户
2. 在左侧编辑区添加片段，输入提示词
3. 右侧实时预览拼合后的完整文本
4. 点击底部「一键复制结果」即可使用

## 使用说明

### 账户与登录

- **注册**：用户名 2-20 字符（支持中文、字母、数字、下划线），密码至少 4 个字符
- **登录**：Token 有效期 7 天，自动加载云端数据
- **修改密码**：右上角用户名 → 修改密码
- **退出登录**：数据保留在服务端，下次登录自动恢复

### 工作区与多标签页

每个工作区是独立的提示词编辑空间，包含自己的片段列表和分隔符设置。

- **创建**：点击标签栏 **+** 按钮，最多 5 个工作区
- **切换**：点击标签页名称
- **重命名**：双击标签页名称
- **排序**：拖拽标签页到目标位置
- **关闭**：点击 **×** 按钮，未保存修改会提示保存
- **重置**：左下角「重置当前工作区」，不影响其他工作区和历史

### 片段编辑

片段（Fragment）是提示词的基本组织单位，可独立启用/禁用、折叠、翻译、着色。

| 操作 | 方法 |
|------|------|
| 新增片段 | 点击 **+ 新增片段组** |
| 编辑标题 | 双击标题文字 |
| 显示标题 | 勾选标题左侧 ☐ 按钮，在输出中包含标题 |
| 启用/禁用 | 点击 👁 眼睛图标，禁用片段不出现在输出中 |
| 折叠/展开 | 点击 ⌃/⌄ 箭头按钮 |
| 批量折叠 | 点击工具栏「全部折叠/还原展开」 |
| 排序 | 拖拽 ⠿ 手柄调整顺序 |
| 删除 | 点击 🗑 按钮（至少保留 1 个） |
| 背景色 | 点击 🎨 调色板，8 种预设色 + 自定义颜色 |

### 标签块模式

每个片段支持两种编辑模式：

| 模式 | 图标 | 特点 |
|------|------|------|
| 标签块模式 | ☰ | 输入自动切分为独立标签块，可单独启停、拖拽排序 |
| 纯文本模式 | ⊞ | 自由输入任意文本，不会被自动切分 |

**标签块操作：**

| 操作 | 方法 |
|------|------|
| 输入标签 | 按 Enter / 逗号 / 中文逗号 / 句号 提交 |
| 编辑标签 | 单击标签 |
| 静音/激活 | 双击标签 |
| 删除标签 | 悬停点击 × 按钮，或输入框空时按 Backspace |
| 拖拽排序 | 拖拽标签到目标位置 |
| 跨片段移动 | 拖拽标签到另一个片段区域 |
| 批量粘贴 | 粘贴多行文本自动按分隔符切分 |

切换回标签块模式时，纯文本会自动按分隔符重新切分。

### 翻译功能

- 点击片段标题栏 **🌐** 图标，一键翻译为英文
- 翻译完成后图标变蓝，**再次点击**切回中文
- 翻译结果缓存在片段中，切换即时，编辑后缓存自动失效

**翻译接口配置**（点击右上角 ⚙ 齿轮图标）：

- **Google 免费直连**（默认）：无需配置，开箱即用
- **自定义 API**：兼容 OpenAI Chat Completions 格式，需填写：
  - 接口标识名称（如 `DeepSeek`）
  - API Base URL（如 `https://api.deepseek.com/v1`）
  - API Key（Bearer Token）
  - Model Name（如 `deepseek-chat`）
- 可添加多个自定义接口，通过单选按钮切换

### 预设库

预设是保存的**单个片段**，可随时追加到任意工作区。

- **存入**：点击片段标题栏 📁 图标
- **使用**：打开预设库 → 点击「追加到当前工作区」
- **管理**：双击重命名，悬停删除

### 快照与历史记录

- **保存快照**：点击 💾 保存图标，输入名称
- **同名冲突**：覆盖 / 自动重命名 / 自行改名
- **加载快照**：历史面板 → 「加载到当前工作区」
- **复制内容**：点击「复制内容」快速复制
- **管理**：双击重命名，悬停删除

### 导入与导出

**导出**：历史面板 → ⬇ 导出 → 勾选导出项（工作区/快照/预设/设置）→ 下载 JSON 文件

**导入**：历史面板 → ⬆ 导入 → 选择 JSON 文件，自动合并（不覆盖已有数据）

### 外观与主题

- 点击右上角 ☀/🌙 图标切换深色/浅色模式
- 默认跟随系统，偏好保存在浏览器本地

### 数据同步

登录后数据自动同步到服务端：

| 数据类型 | 同步延迟 |
|----------|----------|
| 工作区 | 编辑后 ~1 秒 |
| 快照 | 修改后 ~0.8 秒 |
| 预设 | 修改后 ~0.8 秒 |

同步状态显示在工具栏右侧：🔵 同步中 / 🟢 已同步 / 🔴 同步失败

## 部署指南

### Docker Compose（推荐）

```yaml
version: '3.8'

services:
  prompt-builder:
    image: 你的DockerHub用户名/prompt-builder:latest
    container_name: prompt-builder
    ports:
      - "8080:3001"
    volumes:
      - prompt-builder-data:/app/data
    restart: unless-stopped
    environment:
      - PORT=3001
      - DATA_DIR=/app/data
      - NODE_ENV=production
      - JWT_SECRET=你的随机密钥    # ⚠️ 务必修改！
    deploy:
      resources:
        limits:
          memory: 512M

volumes:
  prompt-builder-data:
    driver: local
```

### 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `3001` | 服务监听端口 |
| `DATA_DIR` | `./data` | 数据存储目录 |
| `JWT_SECRET` | `prompt-builder-secret-change-in-production` | JWT 签名密钥，**生产环境务必修改** |
| `NODE_ENV` | - | 设为 `production` 启用静态文件服务 |

### 启动与更新

```bash
# 启动
docker-compose up -d

# 更新
docker-compose pull
docker-compose up -d

# 查看日志
docker logs -f prompt-builder

# 数据备份
docker cp prompt-builder:/app/data ./backup
```

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器（前端热重载）
npm run dev

# 启动后端 API
npm run server

# 构建生产版本
npm run build
```
