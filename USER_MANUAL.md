# Promptly 使用手册

> **Prompt Builder — 提示词拼合器**  
> 一款专为 AI 绘画 / 大模型提示词创作设计的可视化拼合工具。将零散的提示词片段组合、排序、翻译，一键输出完整的提示词文本。

## 1. 快速上手

### 界面布局

Promptly 采用**左右双栏布局**：

| 区域 | 位置 | 功能 |
|------|------|------|
| 标签栏 | 左上方 | 切换/管理工作区标签页 |
| 编辑区 | 左半区 | 编辑提示词片段 |
| 预览区 | 右半区 | 实时查看拼合结果 |
| 工具栏 | 右上方 | 分隔符、翻译设置、主题、保存、历史 |

### 基本流程

1. 登录或注册账户
2. 在左侧编辑区添加片段，输入提示词
3. 右侧实时预览拼合后的完整文本
4. 点击底部「一键复制结果」即可使用

## 2. 账户与登录

### 注册

- 打开应用后，点击**注册**标签页
- 输入用户名（2-20字符，支持中文、字母、数字、下划线）
- 输入密码（至少4个字符）
- 点击「注 册」完成

### 登录

- 输入用户名和密码，点击「登 录」
- 登录后系统自动加载你的云端数据
- Token 有效期为 **7 天**，过期后需重新登录

### 修改密码

- 登录后，点击右上角用户名 → 下拉菜单 → **修改密码**
- 输入旧密码和新密码，确认修改

### 退出登录

- 点击右上角用户名 → 下拉菜单 → **退出登录**
- 退出后数据保留在服务端，下次登录自动恢复

## 3. 工作区与多标签页

### 什么是工作区

每个工作区是一个独立的提示词编辑空间，包含自己的片段列表和分隔符设置。

### 创建工作区

- 点击标签栏右侧的 **+** 按钮
- 最多支持 **5 个**工作区标签页

### 切换工作区

- 点击标签页名称即可切换
- 当前激活的标签页会以高亮样式显示

### 重命名工作区

- **双击**标签页名称，进入编辑模式
- 修改后按 Enter 或点击空白处保存

### 调整标签页顺序

- **拖拽**标签页到目标位置即可排序

### 关闭工作区

- 点击标签页右侧的 **×** 按钮
- 如果工作区有**未保存的修改**（标签上有黄点提示），会弹出确认窗口：
  - 「去保存快照」：先保存再关闭
  - 「不保存，直接关闭」：丢弃修改
  - 「取消」：放弃关闭操作

### 重置工作区

- 点击编辑区左下角「重置当前工作区」
- 确认后清空当前所有片段，恢复默认的 2 个空片段
- **不会影响**其他工作区、历史快照和预设

## 4. 片段编辑

### 什么是片段

片段（Fragment）是提示词的基本组织单位。每个片段可以独立启用/禁用、折叠、翻译、着色。

### 新增片段

- 点击编辑区底部的 **+ 新增片段组** 按钮
- 或从预设库插入（见 [预设库](#7-预设库)）

### 片段标题

- 每个片段顶部有标题栏，默认名称为「片段 N」
- **双击**标题文字可编辑
- 勾选标题左侧的 **☐** 按钮（显示标题），可在最终输出中包含此标题文字

### 启用/禁用片段

- 点击标题栏右侧的 **👁 眼睛** 图标
- 禁用后片段变灰，其内容不会出现在拼合结果中
- 这是临时排除某片段的快捷方式，比删除更安全

### 折叠/展开片段

- 点击标题栏的 **⌃/⌄** 箭头按钮
- 折叠后只显示标题栏，隐藏内容区域
- 顶部工具栏的「全部折叠/还原展开」可批量操作

### 片段排序

- 鼠标悬停在片段标题栏右侧的 **⠿ 拖拽手柄** 上
- 拖拽到目标位置即可调整顺序
- 顺序决定了最终输出的排列顺序

### 删除片段

- 点击标题栏的 **🗑 删除** 按钮
- 最后一个片段无法删除（至少保留 1 个）

### 片段背景色

- 点击标题栏的 **🎨 调色板** 图标
- 选择预设颜色（默认、浅红、浅橙、浅黄、浅绿、浅蓝、浅紫、浅粉）
- 或使用最右侧的 **自定义颜色** 选择器
- 颜色仅用于视觉区分，不影响输出内容

## 5. 标签块模式

### 两种编辑模式

每个片段支持两种编辑模式：

| 模式 | 图标 | 特点 |
|------|------|------|
| 标签块模式 | ☰ | 输入自动切分为独立标签块，可单独启停、拖拽排序 |
| 纯文本模式 | ⊞ | 自由输入任意文本，不会被自动切分 |

点击标题栏的 **☰/⊞** 按钮切换模式。

### 标签块模式操作

#### 输入标签

- 在输入框中输入提示词文字
- 按 **Enter**、**逗号(,)**、**中文逗号(，)** 或 **句号(。)** 提交标签
- 粘贴多行文本会自动按分隔符切分为多个标签

#### 编辑标签

- **单击**标签 → 进入文字编辑模式
- 修改后按 Enter 或点击空白处确认
- 清空文字后自动删除该标签

#### 启停标签（静音）

- **双击**标签 → 切换激活/静音状态
- 静音的标签显示为删除线 + 灰色样式
- 静音标签的内容不会出现在输出中

#### 删除标签

- 鼠标悬停在标签上，点击右上角的 **×** 按钮

#### 拖拽排序

- **拖拽**标签到目标位置调整顺序
- 支持**跨片段拖拽**：将标签从一个片段拖到另一个片段

#### 删除最后一个标签

- 输入框为空时按 **Backspace**，删除最后一个标签

### 纯文本模式

- 切换为纯文本模式后，内容保持为一个完整文本
- 适合需要自由换行和排版的场景（如负面提示词、长描述等）
- 切换回标签块模式时，文本会自动按分隔符重新切分

## 6. 翻译功能

### 一键翻译

- 点击片段标题栏的 **🌐 翻译** 图标
- 将当前片段内容从中文翻译为英文
- 翻译完成后图标变为蓝色高亮
- **再次点击**可切回原始中文

### 翻译缓存

- 翻译结果会缓存在片段中
- 中英文切换是即时的，无需重复调用翻译 API
- 编辑内容后缓存自动失效

### 翻译接口设置

点击预览区右上角的 **⚙ 齿轮** 图标打开翻译设置面板。

#### Google 免费直连（默认）

- 无需配置，开箱即用
- 使用 Google Translate 免费接口
- 可能受网络或频率限制

#### 自定义 API

- 点击「新增自定义 API」添加大模型翻译接口
- 需要填写：
  - **接口标识名称**：如 `DeepSeek`
  - **API Base URL**：如 `https://api.deepseek.com/v1`
  - **API Key (Bearer Token)**：如 `sk-...`
  - **Model Name**：如 `deepseek-chat`
- 接口需要兼容 OpenAI Chat Completions API 格式
- 翻译时系统会发送请求到 `{apiBase}/chat/completions`
- 可添加**多个**自定义接口，通过单选按钮切换
- 点击 **⚙ 展开** 按钮编辑接口配置，**🗑 删除** 按钮移除接口

## 7. 预设库

### 什么是预设

预设是保存的**单个片段**，可以随时追加到任意工作区。适合存储常用的提示词片段（如画风、质量词、场景描述等）。

### 存入预设

- 点击片段标题栏的 **📁 存入预设** 图标
- 空片段无法存入预设（按钮灰色不可点击）

### 打开预设库

- 点击左上方工具栏的 **📚 预设库** 按钮
- 左侧弹出预设面板

### 使用预设

- 在预设库中点击「追加到当前工作区」
- 预设内容会作为新片段添加到当前工作区末尾

### 管理预设

- **重命名**：双击预设标题
- **删除**：鼠标悬停后点击右上角 🗑 图标

## 8. 快照与历史记录

### 保存快照

- 点击预览区右上角的 **💾 保存** 图标
- 输入快照名称（默认使用工作区名称）
- 如果存在同名快照，系统会提示：
  - 「直接覆盖原快照」
  - 「自动重命名并保存为新快照」（自动在名称后追加数字）
  - 「取消，我要自己改名」
- 可使用名称输入框旁的 **+/-** 按钮快速调整数字编号
- 也可点击已有快照名称快速选取

### 查看历史记录

- 点击预览区右上角的 **🔖 历史记录** 图标
- 右侧弹出历史快照面板

### 加载快照

- 在历史面板中点击「加载到当前工作区」
- 快照内容会替换当前工作区的所有片段

### 复制快照内容

- 点击「复制内容」可快速复制快照文本

### 管理快照

- **重命名**：双击快照标题
- **删除**：鼠标悬停后点击右上角 🗑 图标

### 关闭未保存工作区时的提示

关闭有未保存修改的工作区标签时，系统会弹出警告并提供「去保存快照」选项。

## 9. 导入与导出

### 导出数据

1. 打开历史记录面板
2. 点击 **⬇ 导出** 按钮
3. 勾选需要导出的内容：
   - **当前所有工作区**：包括所有标签页内容
   - **历史快照**：保存的快照记录
   - **预设片段库**：预设库中的片段
   - **应用设置**：翻译 API 配置等
4. 点击「确认导出」
5. 文件以 `Prompt_Backup_YYYY-MM-DD.json` 格式下载

### 导入数据

1. 打开历史记录面板
2. 点击 **⬆ 导入** 按钮
3. 选择之前导出的 `.json` 文件
4. 系统自动识别导入内容：
   - 工作区：追加到现有标签页
   - 快照：合并到历史记录
   - 预设：合并到预设库
   - 设置：覆盖翻译配置

> 导入操作为**合并**而非覆盖，已有的数据不会被删除。

## 11. 数据同步

### 自动同步

登录后，所有数据会**自动同步**到服务端：

- **工作区数据**：编辑后 1 秒自动保存
- **快照数据**：保存/修改后 0.8 秒自动同步
- **预设数据**：修改后 0.8 秒自动同步

### 同步状态指示

在左上方工具栏右侧可以看到同步状态：

| 状态 | 图标 | 含义 |
|------|------|------|
| 同步中 | 🔵 旋转图标 | 数据正在上传 |
| 已同步 | 🟢 勾选图标 | 数据已保存到服务端 |
| 同步失败 | 🔴 云断开图标 | 网络异常或服务不可用 |

### 数据存储

- 服务端数据存储在 `DATA_DIR` 目录下（默认 `/app/data`）
- 每个用户有独立的子目录，数据完全隔离
- 密码使用 bcrypt 加密存储，JWT Token 有效期 7 天

## 12. 部署指南

### Docker Compose 部署（推荐）

```yaml
version: '3.8'

services:
  prompt-builder:
    image: 你的DockerHub用户名/prompt-builder:latest
    container_name: prompt-builder
    ports:
      - "8080:3001"
    volumes:
      - prompt-builder-data:/app/data
    restart: unless-stopped
    environment:
      - PORT=3001
      - DATA_DIR=/app/data
      - NODE_ENV=production
      - JWT_SECRET=你的随机密钥    # ⚠️ 务必修改！
    deploy:
      resources:
        limits:
          memory: 512M

volumes:
  prompt-builder-data:
    driver: local
```

### 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `3001` | 服务监听端口 |
| `DATA_DIR` | `./data` | 数据存储目录 |
| `JWT_SECRET` | `prompt-builder-secret-change-in-production` | JWT 签名密钥，**生产环境务必修改** |
| `NODE_ENV` | - | 设为 `production` 启用静态文件服务 |

### 启动与更新

```bash
# 启动
docker-compose up -d

# 更新
docker-compose pull
docker-compose up -d

# 查看日志
docker logs -f prompt-builder

# 数据备份
docker cp prompt-builder:/app/data ./backup
```

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器（前端热重载）
npm run dev

# 启动后端 API
npm run server

# 构建生产版本
npm run build
```
